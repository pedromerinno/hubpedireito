# ADS / Campaign UI Kit

Recreations of Pé Direito's campaign posters, derived from `/ADS/*` and the manifesto / product frames in `/ID/*`.

All four layouts are 1080×1388 (Meta 4:5 feed ad). They share four moves:
1. **Full-bleed poster type** — Tusker Grotesk at 200–420px, always lowercase
2. **Tricolor palette** — flag green, blue, yellow rotate as field / fill / accent
3. **Manifesto voice** — declarative, short lines, "a gente / nota de repúdio"
4. **Flat CTAs** — rectangle, no rounding, inverted brand pair

See `index.html` for:
- A · Green manifesto with yellow tape callout
- B · Yellow field with tricolor overlay diagonals
- C · Black manifesto (rebuke voice)
- D · Product spotlight on cream, with red "LANÇAMENTO" tag
