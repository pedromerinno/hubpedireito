# Instagram UI Kit

Recreations of Pé Direito's Instagram post + story formats, based on `/Insta/*` and `/ID/*_POST, *_STORIES, NOTA_*, S_Alerta_*` in the Figma.

Formats:
- **Feed post** — 1080×1350 (4:5)
- **Story** — 1080×1920 (9:16)
- **Avatar** — 1080×1080 (brand diamond on green)

Layouts covered in `index.html`:
1. **Carrossel cover** — green field, Bayon lead, Tusker poster, yellow diamond
2. **Countdown** — Bayon numerals + blue tape with yellow "FALTAm"
3. **NOTA manifesto** — black field, white Coolvetica copy, the signature rebuke voice
4. **Product story** — yellow field, blue poster type, chinelo render, flat CTA
5. **Alerta de link falso** — blue field, red pill callout, "Não estão 😉"
6. **Cream / blue variant** — the quieter alt palette (see `POST_001`)
