# Site UI Kit — usepedireito.com.br

Recreation of the e-commerce surface based on `/Site/*` (product photography, size tables) and the content vocabulary from `/ID/*`.

Sections in `index.html`:
- **Announcement bar** — blue/yellow, uppercase Bayon
- **Nav** — Bayon links, wordmark centered, sacola CTA in yellow
- **Hero** — green field, huge Tusker poster "pé direito.", yellow CTA
- **Coleção grid** — four colorways (Azul, Amarelo, Verde, Branco), gradient surfaces keyed to the product color
- **Manifesto band** — black field, "nota de repúdio" voice
- **Footer** — blue field, yellow text, three-column link list

Note: this site was not fully designed in the Figma — only flat product photography and a size table exist under `/Site/`. This kit **reasonably extrapolates** the e-commerce surface from the brand system (colors, type, voice) established across ID/Insta/ADS. Flag for the user if they want this grounded in a real site design.
